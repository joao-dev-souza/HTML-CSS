Outras formatações

Codigo-fonte/Pré-formatação

podemos usar o codigo fonte para deixar todos os espaços deixados para letras com o mesmo tamanho e é a fonte usada em codigos como por exemplo <script>document.write(window.location.href);

Pré formatação é usada para ler tudo o que está dentro e considerar como por exemplo espaços, pulos de linha e etc...

podemos mesclar os dois assim:

        
<script>
        var str = 'Terminal Root';

        if(str.match(/Root/)){
            document.write('Encontrou');
        }else{
            document.write('Palavra Não Encontrada');
        }
        </script>
